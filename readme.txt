Logging and Tracking your User visit History

== Description ==

User Page Logs help you to Tracking and logging your visitor page view. you can also view detailed information about their page view and with ability to export it to CSV. Want to know what pages your user visited? this plugin is your solution. all page that your user visited will be recorded and showed to you in wp admin UPL dashboard page.
